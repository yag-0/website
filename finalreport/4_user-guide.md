## Функції системи

### 1. Каталог товарів

#### Перегляд товарів
1. Натисніть "⛁ Каталог" або `Alt + M`
2. Перегляньте список доступних товарів
3. Кожна картка товару містить:
   - Фото товару
   - Назву
   - Короткий опис
   - Ціну
   - Наявність на складі
   - Кнопку "Додати в кошик"

**Приклад картки товару:**
```
┌────────────────────────────────┐
│  [Фото ноутбука]              │
│                                │
│  Ноутбук ASUS TUF Gaming      │
│  Потужний ігровий ноутбук     │
│                                │
│  25,999.99 грн                │
│  В наявності: 15 шт            │
│                                │
│  [Додати в кошик]             │
└────────────────────────────────┘
```

#### Фільтрація товарів
- Використовуйте випадаюче меню "Категорія" для фільтрації
- Доступні категорії:
  - Усі товари
  - Ноутбуки
  - Периферія
  - Аксесуари
  - Комплектуючі

#### Пошук товарів
1. Введіть назву товару в поле пошуку
2. Натисніть "Шукати" або Enter
3. Система відобразить відповідні результати

---

### 2. Кошик покупок

#### Додавання товарів
1. На сторінці каталогу натисніть "Додати в кошик"
2. Товар з'явиться в кошику
3. Кількість товарів у кошику відображається поряд з іконкою 💭

#### Перегляд кошика
1. Натисніть "💭 Кошик" або `Alt + C`
2. Переглядайте список обраних товарів
3. Для кожного товару доступно:
   - Зміна кількості (+/-)
   - Видалення з кошика (🗑️)

**Приклад кошика:**
```
┌────────────────────────────────────────────────┐
│  Ваш кошик                                     │
├────────────────────────────────────────────────┤
│  Ноутбук ASUS TUF                             │
│  Кількість: [−] 1 [+]        25,999.99 грн    │
│  [🗑️ Видалити]                                 │
├────────────────────────────────────────────────┤
│  Миша Logitech G502                           │
│  Кількість: [−] 2 [+]         2,998.00 грн    │
│  [🗑️ Видалити]                                 │
├────────────────────────────────────────────────┤
│  Загальна сума:              28,997.99 грн    │
│                                                │
│  [Очистити кошик] [Оформити замовлення]       │
└────────────────────────────────────────────────┘
```

#### Оформлення замовлення
1. У кошику натисніть "Оформити замовлення"
2. Заповніть форму:
   - Ім'я та прізвище
   - Email
   - Номер телефону
   - Адреса доставки
   - Промокод (опціонально)
3. Натисніть "Підтвердити замовлення"
4. Отримаєте підтвердження з номером замовлення

**Форма замовлення:**
```
┌────────────────────────────────────┐
│  Оформлення замовлення             │
├────────────────────────────────────┤
│  Ім'я: [________________]          │
│  Email: [________________]         │
│  Телефон: [________________]       │
│  Адреса: [________________]        │
│  Промокод: [________]  [Перевірити]│
├────────────────────────────────────┤
│  Загальна сума: 28,997.99 грн      │
│  Знижка: 0.00 грн                  │
│  До сплати: 28,997.99 грн          │
│                                    │
│  [Підтвердити замовлення]          │
└────────────────────────────────────┘
```

### 3. Промокоди

#### Використання промокоду
1. При оформленні замовлення введіть промокод
2. Натисніть "Перевірити"
3. Якщо промокод дійсний:
   - Відобразиться % знижки
   - Загальна сума перерахується автоматично
4. Якщо промокод недійсний:
   - З'явиться повідомлення про помилку

#### Приклади промокодів
- `WINTER25` - 25% знижка до 28.02.2025
- `SPRING20` - 20% знижка до 31.05.2025
- `FIRST10` - 10% знижка для нових клієнтів

**Приклад застосування знижки:**
```
Сума без знижки:  28,997.99 грн
Промокод WINTER25 (25%):  -7,249.50 грн
──────────────────────────────────
До сплати:        21,748.49 грн
```

## Поширені питання

### Загальні питання

**Q: Як перевірити статус замовлення?**
A: Після оформлення замовлення ви отримаєте номер замовлення на email. Зв'яжіться з підтримкою для уточнення статусу.

**Q: Чи можна скасувати замовлення?**
A: Так, зв'яжіться з підтримкою протягом 24 годин після оформлення.

**Q: Скільки часу триває доставка?**
A: Звичайна доставка: 2-5 робочих днів. Експрес-доставка: 1-2 робочі дні.

**Q: Які способи оплати доступні?**
A: Оплата при отриманні, банківська картка, PayPal, Google Pay.

### Технічні питання

**Q: Чому не працює промокод?**
A: Перевірте:
- Правильність написання (враховується регістр)
- Термін дії промокоду
- Умови використання (мінімальна сума замовлення)

**Q: Чому товар не додається в кошик?**
A: Можливі причини:
- Товару немає в наявності
- У браузері відключені cookies
- Технічна проблема - спробуйте оновити сторінку

**Q: Чому не відображаються зображення товарів?**
A: Спробуйте:
- Очистити кеш браузера (Ctrl + F5)
- Перевірити інтернет-з'єднання
- Використати інший браузер

**Q: Як експортувати дані про замовлення?**
A: Використовуйте API endpoint `/api/orders` для отримання даних у JSON форматі.

## Вирішення проблем

### Проблема 1: Сайт не відкривається

**Симптоми:**
- Помилка "Неможливо підключитися до сервера"
- Пуста сторінка

**Рішення:**
1. Перевірте, чи запущений сервер
2. Переконайтеся, що використовуєте правильний URL (http://localhost:5000)
3. Перевірте, чи не блокує з'єднання фаєрвол
4. Спробуйте інший браузер

### Проблема 2: Кошик очищується після перезавантаження сторінки

**Причина:** Проблема з cookies або session storage

**Рішення:**
1. Увімкніть cookies в налаштуваннях браузера
2. Не використовуйте режим інкогніто
3. Очистіть дані сайту і спробуйте знову

### Проблема 3: Гарячі клавіші не працюють

**Можливі причини:**
- Активне поле вводу (курсор в текстовому полі)
- Конфлікт з розширеннями браузера

**Рішення:**
1. Натисніть поза полями вводу
2. Спробуйте відключити розширення браузера
3. Використовуйте кнопки навігації замість клавіш

### Проблема 4: Темна тема не зберігається

**Рішення:**
1. Перевірте, чи дозволені localStorage в браузері
2. Не використовуйте режим інкогніто
3. Очистіть localStorage: F12 → Application → Local Storage → Clear

### Проблема 5: Повільна робота сайту

**Рішення:**
1. Перезавантажте сторінку (F5)
2. Очистіть кеш браузера (Ctrl + Shift + Del)
3. Закрийте непотрібні вкладки
4. Перевірте швидкість інтернету
5. Перезапустіть браузер
